# Pryzen Portfolio

Personal portfolio website for Pryzen — interactive splash screens, project showcase, certificates, and light/dark mode.

## Pages

- `index.html` — Home
- `portfolio.html` — Portfolio

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```

## Stack

HTML, CSS, JavaScript (GSAP animations)
